Estrutura do formulário

Podemos criar um formulário usando o elemento <form>, que deve sempre ter o atributo action. O valor do atributo action é a URL da página no servidor que receberá as informações do formulário quando ele for submetido.

Atributos do formulário

O atributo action define a ação que deve ser executada quando o formulário é enviado. Normalmente, os dados do formulário são enviados para um arquivo no servidor quando o usuário clica no botão enviar.

Imagens

Para adicionar uma imagem na página web nós empregamos o elemento <img>, que é um elemento vazio (não tem tag de fechamento).

O elemento <img> tem dois atributos básicos que são requeridos:

  • src (source): especifica o caminho para a imagem.
  • alt (alternative): especifica um texto alternativo para a imagem, que serve caso o usuário, por algum motivo, não puder visualizá-lo
    (devido à conexão lenta, um erro no atributo src ou se o usuário usar um leitor de tela).

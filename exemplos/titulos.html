AVALIANDO A HIERERQUIA DOS TÍTULOS EM HTML

Os títulos em HTML possuem SEIS NÍVEIS de hierarquia, que são definidos com as tags <h1> a <h6>. Usamos o <h1> para mostrar qual é o assunto principal que será tratado.

Este é um sub-assunto do título h1;

Quando formos falar sobre um sub-assunto do assunto principal, usamos o <h2>.

Este é um sub-assunto do título h2

Quando formos falar sobre um sub-assunto do sub-assunto secundário, usamos o <h3>

Este é um sub-assunto do título h3

Quando formos falar sobre um sub-assunto do sub-assunto terciário, usamos o <h4>.

Este é um sub-assunto do título h4

Quando formos falar sobre um sub-assunto do sub-assunto quaternário, usamos o <h5>.

Este é um sub-assunto do título h5

Quando formos falar sobre um sub-assunto do sub-assunto quinternário, usamos o <h6>.
